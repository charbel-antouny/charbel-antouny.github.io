Thesis Project

The Honours component of my degree required me to complete a thesis project, and I chose to redesign the UNSW handbook. The handbook provides information about all the programs, streams and courses offered by the university. The most notable flaw of the current implementation is that it is not synchronised with the university’s database, that is, the data on each page is statically inserted. My implementation solves this issue using a PostgreSQL and PHP back end to pull the data, and HTML and JavaScript on the front end to display it. Most of the CSS is handled by the Bootstrap framework. My project received a distinction grade and is available to view (desktop only) by clicking the subheading.
